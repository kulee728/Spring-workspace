2. thymeleaf 

		
			> 웹 및 독립 실행형 환경 모두를 위한 최신 서버쪽 Java 템플릿 엔진
			> 웹 실행 == 요청 시 포워드 되는 화면
			>> 독립 실행 == html 파일 실행 (인터넷 연결 없이)
			
			HTML 파일에서 th (ThymeLeaf)속성을 이용해 컨트롤러부터 전달받은 데이터를
			이용, 동적 페이지를 이용할 수 있음.
			Spring boot 에서는 jsp 아닌 ThymeLeaf 사용을 권장하고 있음.
			
			[사용법]
			html 에서
			!Doctype html 보다 앞에 작성
			태그열기 html xmlns:th="http://www.thymeleaf.org" 태그닫기 작성 후 사용할 수 있다.
			http://www.thymeleaf.org 에 대한 별칭 설정
			html xmlns:별칭 = "템플릿 가져올 주소"
